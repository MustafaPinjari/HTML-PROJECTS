# 🔋 Battery Detector | Mustafa Pinjari

Welcome to the Battery Detector project! This is a simple web application that detects and displays the current battery level and charging status of your device using HTML, CSS, and JavaScript.

## 📋 Project Overview

This project visualizes the battery level and charging status of your device in a creative and interactive way. It's a great example of how to use the Battery Status API in a web application.

![Battery Detector Screenshot](screenshot.jpg) <!-- Add a screenshot of your project here -->

## 🌟 Features

- 🚀 Real-time battery level detection.
- ⚡ Displays charging status with animations.
- 🎨 Dynamic battery level indicator with color gradients.
- 🖥️ Responsive design suitable for all devices.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites

- A web browser (Chrome, Firefox, Safari, etc.)

### Installation

1. **Clone the repository**:
   ```sh
   git clone https://github.com/MustafaPinjari/HTML-PROJECTS/Battery-Detector.git
   ```
2. **Navigate to the project directory**:
   ```sh
   cd Battery-Detector
   ```
3. **Open the project**:
   - Open `index.html` in your favorite web browser or use a live server extension in your code editor.

## 🖥️ Usage

- Simply open the `index.html` file in a web browser.
- The battery level and charging status will be displayed automatically.
